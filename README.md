## What is MobAI?

MobAI gives AI coding agents like Claude Code, Cursor, and Codex the ability to see and control Android & iOS devices. It provides:

- **MCP Server** - Native Model Context Protocol integration
- **HTTP API** - Simple REST API for any AI agent
- **Visual Feedback** - Screenshots, UI trees, and device control
